How to read the value table

Each row shows a pet's Regular, Neon, and Mega tier values from two community sources: Elvebredd (E) and AMVGG (A). The two numbers in each cell represent the same pet on two different value lists. When both sources agree, that value is well-established by community consensus. When they differ, the spread is informative — it tells you the range of values you might encounter when trading. A delta (Δ) badge next to a value means that pet's price changed since the last hourly scrape. Click any row to view the pet's full value history chart and use the trade calculator for that pet directly.

What do the rarity tiers mean?

Adopt Me classifies every pet into one of five rarity tiers. Rarity generally correlates with trade value, though demand and limited availability can push even lower-rarity pets above their tier's typical range.

  • Common — Easily obtained from standard eggs. Low trade value. Examples: Cat, Dog, Chicken.
  • Uncommon — Slightly harder to find. Minor value above Common. Examples: Horse, Bunny.
  • Rare — Noticeably harder to obtain. Meaningful trade value. Examples: Beaver, Rabbit.
  • Ultra-Rare — Difficult to obtain. Significant value. Examples: Hyena, Wolf, Platypus.
  • Legendary — The hardest to obtain, highest trade value. Legendary pets like the Frost Dragon, Shadow Dragon, and Giraffe are among the most traded in the game.

Understanding the delta (Δ) indicator

The delta badge shows how much a pet's value has changed since the previous hourly scrape. A positive delta (green, upward) means the value increased on that source since the last update. A negative delta (red, downward) means it decreased. Pets with active delta badges are currently experiencing value movement — useful to know before entering a trade involving that pet. Use the "Changed only" filter to see only pets with recent value changes.
