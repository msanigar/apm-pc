About Pet Values

All values are measured in Ride Potions (RP), which serves as the standard currency unit for Adopt Me trading values.

Neon pets are worth approximately 4x the regular value, while Mega Neon pets are worth approximately 16x the regular value.

Fly (F) and Ride (R) potions add small value bonuses. FR pets are generally the most desirable variant.
